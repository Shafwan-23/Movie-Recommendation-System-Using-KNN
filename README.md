# 🎬 Movie-Recommendation-System-Using-KNN

This is a Flask-based web application that provides personalized movie recommendations using a K-Nearest Neighbors (KNN) machine learning model. Users can register, log in, like movies, search by title, filter by genres, and view tailored recommendations based on their preferences.

---

## 🚀 Features

- ✅ User registration and login system with secure password hashing (bcrypt)
- 🎯 Personalized movie recommendations using KNN
- ❤️ Like movies to improve your recommendation results
- 🔍 Search for movies by title
- 🎭 Filter recommendations by selected genres
- 👤 Profile page displaying all liked movies
- 📊 Integration with machine learning models using `joblib` and `pandas`
- 🗃️ Backend powered by MySQL

---

## 🧰 Technologies Used

- **Framework**: Flask (Python)
- **Database**: MySQL with Flask-MySQLdb
- **Frontend**: HTML, CSS, Bootstrap, Jinja2
- **Machine Learning**: scikit-learn (KNN), pandas, numpy
- **Security**: bcrypt for password hashing
- **Model Storage**: `joblib` for serialized ML models
